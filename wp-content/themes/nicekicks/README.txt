<h2>Instructions</h2>

<strong>Menu.</strong> To edit the site's menus, go to Appearance > Menus and select the appropriate menu at the top. <a href="/wp-admin/index.php?page=wp101&document=13">Video on Menus.</a>

<strong>Pages.</strong> To edit a page's content, go to Pages and select the appropriate page. You can insert an image using the first icon next to Upload/Insert. <a href="/wp-admin/index.php?page=wp101&document=5">Video on Creating and Editing Pages.</a>

<strong>Posts.</strong> To edit the posts in your blog, go to the Posts section. <a href="/wp-admin/index.php?page=wp101&document=2">Video on Creating Posts</a>

<strong>Sidebar.</strong> Customize the sidebar widgets from Appearance > Widgets. <a href="/wp-admin/index.php?page=wp101&document=12">Video on Adding Widgets.</a>
